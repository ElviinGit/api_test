📖 Overview

This repository demonstrates how to build an API testing framework in Python using pytest and requests.
The goal is to showcase my ability to design, organize, and execute automated tests for REST APIs — covering CRUD operations (Create, Read, Update, Delete) and common validation patterns.

I use public, free APIs (like JSONPlaceholder) so anyone can run the tests without authentication or setup.

🎯 Objectives

✅ Show understanding of REST API testing principles

✅ Implement a clean, reusable test framework with pytest

✅ Cover CRUD operations with assertions on both status codes and response bodies

✅ Demonstrate fixtures, modular test design, and reporting

✅ Provide clear documentation so reviewers can easily run and evaluate the project

🛠 Tech Stack

Python 3.x

pytest → test runner

requests → HTTP client

pytest-html → optional HTML reporting
